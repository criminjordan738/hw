1. 冷静期结束后，进入欧易 APP 的“资产”页面，选择“提币”。  
2. 提币地址填写 Depay 的充值地址，主网选择 **USDT-TRC20**，填写提币金额并提交。  
3. 回到 Depay APP，将充值到账的 USDT 兑换为 USD，然后充值到虚拟信用卡。
